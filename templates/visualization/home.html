About This Project

AlgoViz Pro is a comprehensive algorithm visualization platform built for the INF601 Advanced Programming with Python course. It demonstrates:

  • Modular Django application architecture (4 separate apps)
  • RESTful API integration with GitHub
  • Real-time data visualization using HTML5 Canvas
  • Code complexity analysis using Python's AST module
  • Comprehensive error handling and caching strategies
  • Professional documentation and testing practices
